# 🤖 ربات کیوریتور خودکار محتوای هوش مصنوعی

ربات تلگرامی حرفه‌ای برای جمع‌آوری، پردازش و ارسال خودکار محتوای هوش مصنوعی از منابع مختلف.

## ✨ قابلیت‌ها

- **جمع‌آوری از ۸ منبع**: RSS، کانال تلگرام، توییتر، Reddit، وبسایت، گیت‌هاب، Hacker News، arXiv
- **پردازش هوشمند**: خلاصه‌سازی فارسی، دسته‌بندی، امتیازدهی با AI
- **فیلتر پیشرفته**: کلمات کلیدی مثبت/منفی، دسته‌بندی، حداقل امتیاز
- **ارسال زمان‌بندی شده**: خلاصه روزانه + ارسال فوری مطالب مهم
- **پنل مدیریت کامل**: همه چیز از داخل خود ربات

## 🛠️ پیش‌نیازها

- Python 3.10+
- حداقل یک کلید API هوش مصنوعی (OpenAI، Claude، یا DeepSeek)
- توکن ربات تلگرام (از @BotFather)
